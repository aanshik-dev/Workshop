## ЁЯРжтАНЁЯФе Arduino Ide

тЪб Requirement : Arduino IDE 1.8.19

тЪб Installation:

- Link: `https://www.arduino.cc/en/main/software`
- Click on Windows Installer
- Run the exe file
- Click on last Checkbox: `Add Arduino IDE to PATH`
- Click on Install Now
- Open Terminal to varify installation
- `arduino --version` // should return `1.8.19`

тЪб Requirement : Python 3.11.0

тЪб Installation:

- Link: `https://www.python.org/downloads/release/python-3110/`
- Click On Windows Installer
- Run the exe file
- Click on last Checkbox: `Add Python to PATH`
- Click on Install Now
- Open Terminal to varify installation
- `python --version` // should return `Python 3.11.0`
